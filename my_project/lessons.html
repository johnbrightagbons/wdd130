Lesson 1


DEFINITION OF COMPUTER

Computer is defined as an electronic device capable of accepting data, processing data, storing data and producing an output called information following a set of instructions called program. It is also seen as an electronic device for storing and processing data, typically in binary form, according to instructions given to it in a variable program.
A computer is a digital electronic machine that can be programmed to carry out sequences of arithmetic or logical operations automatically. Modern computers can perform generic sets of operations known as programs. These programs enable the computer to do a variety of tasks.

PARTS OF A COMPUTER
The basic parts of a computer system are:
1. System Unit
2. Monitor
3. Keyboard
4. Mouse
5. Speaker

CHARACTERISTICS OF A COMPUTER
The Characteristics of computer include:
1. Accuracy: The results produced by computers are very correct.
2. Diligence: Computers are free from tiredness, dullness and lack of concentration.
3. Large Storage Capacity: Computers can store very large amount of data.
4. Speed: Computers process data at a very high speed.
5. Automatic: A device is said to be automatic if it can complete tasks on it own. Computers once started on a task can complete such task without human intervention.
6. Versatility: Computers can be used to perform different tasks.

Lesson 2


DATA AND INFORMATION

What is data? Data is a raw fact or observation about a person, place, thing or event. Data is an unorganized fact, a fact that has not been worked upon or processed. Data is meaningless, and may not make a complete sense. Data is unreliable as its context can change. Examples of data are alphabets (A, B, C or a, b, c), numbers (0 - 9), characters, symbols etc.
What is information? Information is a processed fact. It a data that has been converted into a meaningful context, and therefore makes a complete sense. It can also be seen as an organized data. Examples of information are, the total number of people in Nigeria, the total number of girls in a school, the result of an exam, the number of workers in a factory etc.

DIFFERENCE BETWEEN DATA AND INFORMATION
1. Data is a raw fact while Information is a processed data
2. Data is unreliable while information is reliable
3. Data is not significant but information is significant
4. Data cannot be used for decision making but information can be used
5. Data is meaningless but information is meaningful
6. Data is unorganized while information is organized

Lesson 3


DATA PROCESSING

Meaning of data procesing
Data processing can be defined as the process of gathering data from various sources performing some required actions on them in order to convert them into meaningful context for decision making. It simply means the process of changing data into informaation. Before data is used for making any decision such data need to be converted into a meaningful context, the process of converting such data to information is termed data processing.

STAGES OF DATA PROCESSING
The following are stages or cycles of data processing;
1. Data gathering or data collection stage
2. Data preparation stage
3. Input stage
4. Processing stage
5. Output stage
6. Storage stage

IMPORTANCE OF THE COMPUTER AS A PROCESSING TOOL
The importance of computer as a processing tool include;
1. Increased accuracy
2. Efficient storage Capacity
3. Easy retrieval of information
4. Fast access to information
5. Handle difficult tasks
